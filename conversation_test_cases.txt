Movie CRS Conversation Test Cases

Purpose:
Reference-style multi-turn conversations for testing both recommender paths.
The format follows data/llm_redial/Conversation.txt: numbered conversations
with User and Agent turns.

Notes:
- Recommendation turns use the API prompt format: **Title** - reason.
- Chit-chat, clarification, and closing turns should not produce parsed recommendations.
- Old-user cases use real LLM-Redial user ids from data/llm_redial/user_ids.json.
- These are golden-style reference transcripts, not guaranteed byte-for-byte LLM output.
